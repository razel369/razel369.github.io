Evidence Bundle חתום

ZIP שמכיל את כל ההיסטוריה, חתום ב-Ed25519. הרגולטור יכול לאמת את החתימה בלי להתקין שום דבר.

⚡

30 שניות להתקנה

3 שורות Python. אפס תלויות בכבדות. PostgreSQL או SQLite. קוד פתוח (MIT). אפשר לעשות self-host מאחורי ה-VPC שלכם.

🔍

Re-evaluate על היסטוריה

מדיניות השתנתה? הריצו אותה על כל ההחלטות שכבר נעשו. אידיאלי לבדיקות עומק לפני ביקורת רגולטורית.

ככה זה נראה

התקנה, הפעלה, וקבלת evidence bundle — הכל בפחות מדקה.

# 1. מתקינים pip install ai-integrity # 2. מוסיפים 3 שורות לקוד שלכם from ai_integrity import AgentClient client = AgentClient(api_key="aik_live_...") @client.track(tool="credit_decision") def process_loan(application): result = my_llm.analyze(application) client.record_event(tool="credit_decision", payload=result) return result # 3. מקבלים evidence bundle חתום curl https://api.ai-integrity.dev/v1/runs/run_8a3f2c/evidence.zip -H "X-API-Key: aik_live_..." -o evidence.zip # ה-ZIP מכיל: # ✓ audit.json — כל האירועים, hash-chained # ✓ audit.txt — סיכום לרגולטור # ✓ summary.md — הסבר אנושי # ✓ chain_proof.txt — הוכחת tamper-evidence # ✓ signature.sig — חתימת Ed25519
